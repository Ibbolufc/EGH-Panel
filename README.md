# EGH Panel

EGH Panel is a self-hosted game server management panel with a custom node agent called **EGH Node**.

This repository contains the **panel** application.  
The **node daemon** lives in the separate `EGH-Node` repository.

---

## What currently works

- Fresh panel install with a one-line bootstrap command
- Fresh node install with a one-line command generated by the panel
- Custom `EGH Node` binary download via the panel
- Node configuration written automatically
- `systemd` service created automatically
- Node heartbeat back to the panel
- Node connection testing from the panel
- Clean fresh-install flow for new users

---

## Repositories

- **EGH Panel:** this repository
- **EGH Node:** `Ibbolufc/EGH-Node`
